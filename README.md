# Anime Verse Shop

Большой Telegram Mini App-проект в аниме-стилистике с главным акцентом на магазин персонажей и одежды.

## Концепция

Пользователь выбирает парня или девушку, собирает образ и покупает одежду, причёски, аксессуары, оружие, питомцев и визуальные эффекты за внутриигровую валюту — кроны.

## Стек

- React
- TypeScript
- Vite
- FastAPI
- Telegram Mini Apps
- Vercel
